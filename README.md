App acabada, para usar:
  - En el backend, en application.properties poner datos de la base de datos del servidor (username,password...)
  - En el frontend, en retrofitClient, las 2 url estaticas tienes una ip, para usar en moviles físicos poner la ip del servidor, debe estar en la misma red que los dispositivos que usan la app.

Para ejecutar:
  - Abrir MySQL
  - Abrir el backend en un IDE adecuado como IntelliJ/Eclipse/Spring Boot y ejecutarla.
  - Abrir el frontend en Android Studio o desde un dispositivo con la app instalada y ejecutarla con el servidor (backend) ejecutado.
