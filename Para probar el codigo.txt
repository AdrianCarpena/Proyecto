Para el servidor. Cambiar el username y password al de tu base de datos en:
Servidor/backend/src/main/resources/application.properties
	spring.datasource.username=root
	spring.datasource.password=root

Aplicacion/app/src/main/java/com/example/proyecto/api/RetrofitClient.java

	private static final String BASE_URL = "http://10.0.2.2:8080/";
